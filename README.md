# imieninyApp
Aplikacja która pokazuje czyje są imieniny i nietypowe święta, można w niej zapisywać przypomnienia i sprawdzać kiedy kot ma imieniny. Scrapuje ze strony Imienniczek.pl i  kalibi.pl
